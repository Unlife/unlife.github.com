Опыт разработки

Создание шаблона сайта предприятия "с нуля", начиная от отрисовки эскиза в Photoshop CS5, до верстки в Dreamviewer CS3.
Создание данного сайта-резюме с нуля.
Также был создан второй тестовый сайт-резюме с помощью CMS Joomla 2.5, за основу был взят шаблон joomla, который был значительно переделан (перерисовка меню, нарезка для слайдшоу хэдера, правка CSS файлов).

Языки программирования

HTML и CSS, адаптация готовых jQuery скриптов для встраивания в сайт.
HTML используется для создание каркаса сайта, CSS для разметки и оформления, Jquery для добавления эффектов.
Для оформления простых элеметов навигации предпочитаю использовать спрайты, для более сложных - Jquery, однако применять javascript предпочитаю умеренно, чтобы не иметь проблем с кроссбраузерностью.

Редакторы и среды

  • Создание эскиза, отрисовка кнопок и спрайтов, а также элементов оформления и навигации выполняется в Adobe Photoshop CS5 ;
  • Для верстки сайта используется Adobe DreamViwer CS3 ;
  • Просмотр и правка html, css, js и php файлов выполняется при помощи редактора Notepad++
  • Для создания сайта из готового шаблона используется joomla 2.5 и пакет локального сервера denwer ;
  • Для анализа страницы используется встроенный в Google Chrome редактор страниц или же плагин FireBug для Mozilla Firefox.

    • Использование систем

    По опыту работы использовал практически все ОС семейства Windows, включая серверные, цели - самые различные, от организации рабочего места, до развертывания DNS, DHCP, терминальных серверов.

    Знаком с системой контроля версий Git, однако в данной работе не использовал. С системами постановки и ведения задач я не сталкивался.
